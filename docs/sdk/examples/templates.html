Template Plugin Documentation

The Content Templates plugin provides a dialog to offer a choice of predefined document templates - with page layout, text formatting and styles. It is included in the Full distribution.

It allows to apply predefined formatting templates to the document content.

When enabled, the plugin adds the Templates () toolbar button which lets the user invoke a dialog with defined templates.
